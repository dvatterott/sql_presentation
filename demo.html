Why a data-pipeline?

At Showtime, deriving data-driven insights about our customers required dedicated effort.

The data-science team built the data-pipeline so that these insights are instantly available to the entire business.

What is hard about designing a data-pipeline?

  • It should provide important and otherwise unavailable data.
  • It should be extensible and flexible.
  • It should be scalable.

function linkify( selector ) {
  if( supports3DTransforms ) {

    var nodes = document.querySelectorAll( selector );

    for( var i = 0, len = nodes.length; i < len; i++ ) {
      var node = nodes[i];

      if( !node.className ) {
        node.className += ' roll';
      }
    }
  }
}
